# datasciencecca
**🚗 Used Car Price Predictor**

A Machine Learning web application that predicts the resale price of used cars based on key features like fuel type, ownership, transmission, and more.

**Model Information**
Algorithm Used: 
KNeighborsRegressor (KNN-3)

**Features Used:**
Insurance Validity
Fuel Type
Kilometers Driven
Ownership
Transmission
Year
Mileage
Engine
Power
Torque
Seats

**✨ Features**
Clean and user-friendly UI
Dropdown-based inputs (no invalid entries)
Instant price prediction
Lightweight and fast

**🛠️ Tech Stack**
Python
Streamlit
Scikit-learn
NumPy

**⚙️ Run Locally**
pip install -r requirements.txt
streamlit run app.py
